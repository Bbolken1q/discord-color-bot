## Discord bot 

Simple discord bot for my friend's server written in Rust with [Serenity](https://github.com/serenity-rs/serenity)

Features: 
* Custom color roles
* Replying to specified phrases
* Configuration panel 

Commands: 
`/color [param: color in RGB/Hex]`
Sets username color to specified RGB (formatted like "rgb(x, y, z)") or Hex (formatted like "#xyzabc") value
